## Инструкция сборки  
Перейдите в папку TESTS и создайте папку buid для сборки:  

mkdir build  

Перейдите в папку build:  

cd build  

Выполните ряд следующих команд для сборки:  

cmake ..  
make 
## Инструкция использование
Запустите исполняемы файл:  
./runTests
